# 🗳️ Online Voting System

A simple and interactive **Online Voting System** developed using **PHP, HTML, CSS, and JavaScript**. This project allows users to vote for their preferred candidate, calculates vote percentages, determines the winner, and displays voting results dynamically.

## 📌 Features

* Vote for a candidate using radio buttons
* One vote per user session
* Dynamic vote counting
* Automatic percentage calculation
* Winner determination
* Live voting results
* Progress bars for vote visualization
* Responsive and user-friendly interface
* Reset voting functionality
* Modern dashboard-style UI

## 🛠️ Technologies Used

* PHP
* HTML5
* CSS3
* JavaScript
* PHP Sessions

## 🚀 How to Run

### Using XAMPP

1. Install XAMPP.
2. Start Apache from the XAMPP Control Panel.
3. Copy the project folder into:

```text
xampp/htdocs/
```

4. Open your browser and visit:

```text
http://localhost/Online-Voting-System/
```

5. Start voting.

## ✨ System Workflow

1. User selects a candidate.
2. User submits their vote.
3. Vote count is updated.
4. Session prevents multiple voting.
5. System calculates:

   * Total Votes
   * Vote Percentage
   * Election Winner
6. Results are displayed dynamically.
